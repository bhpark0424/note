# Linux Userspace Library (SystemCall / Standard C Library)

## Linux Userspace Library
### fopen
```c
FILE *fopen(const char *path, const char *mode);
```
- 입력
    - path : 파일 경로
    - mode : read/write/append 등의 설정
        - "r", "r+", "w", "w+", "a", "a+"
- 리턴
    - 파일 포인터
- 기능 설명
    - 파일을 다루기 위한 파일 포인터를 얻기 위한 단게. 보통 파일을 "연다(open)" 라고 함
